API Gateway 7.7 Apache Cassandra Administrator Guide

Apache Cassandra is a highly scalable distributed NO-SQL database that provides high service availability. It is required to store data for API Manager or API Gateway client registry. In addition, Cassandra is optional to store data for Custom KPS table definitions and data and OAuth token stores API Gateway components.

This guide explains how to configure and manage the Apache Cassandra database for API Gateway and API Manager.
